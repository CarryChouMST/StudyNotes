## iFlow-Cli

- iflow的升级

  - 卸载当前版本的iflow-cli：

    ~~~cpp
    npm uninstall -g @iflow-ai/iflow-cli
    ~~~

  - 使用以下命令安装最新版本：

    ~~~cpp
    bash -c "$(curl -fsSL https://cloud.iflow.cn/iflow-cli/install_beta.sh)"
    ~~~

  - 如果您不想卸载当前版本，可以直接使用以下命令进行升级：

    ~~~cpp
    bash -c "$(curl -fsSL https://cloud.iflow.cn/claude-code/install.sh)"
    或者
    npm i -g @ali/claude-code --registry=https://registry.anpm.alibaba-inc.com
    ~~~
