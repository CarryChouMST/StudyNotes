## iFlow-Cli

- iflow的升级

  - 卸载当前版本的iflow-cli：

    ~~~cpp
    npm uninstall -g @iflow-ai/iflow-cli
    ~~~

  - 使用以下命令安装最新版本：

    ~~~cpp
    bash -c "$(curl -fsSL https://cloud.iflow.cn/iflow-cli/install_beta.sh)"
    ~~~

  - 如果您不想卸载当前版本，可以直接使用以下命令进行升级：

    ~~~cpp
    bash -c "$(curl -fsSL https://cloud.iflow.cn/claude-code/install.sh)"
    或者
    npm i -g @ali/claude-code --registry=https://registry.anpm.alibaba-inc.com
    ~~~

- AK的API密码
  - 可以在心流官网中以个人账号申请|对于内部员工，有内网官网xinliu.alibaba-inc.com
  - 也可以在IdeaLab中我的管理下，去以集团身份申请key。|目前对于IdeaLab的申请不完善
  
- 注意一般互联网服务对个人使用有一个官网，对于api使用会有一个平台官网，在心流开放平台下有配置的一些教程https://platform.iflow.cn/cli/configuration/settings

- 注意base-url后面要有斜杠，用于后面申请的拼接

    "selectedAuthType": "iflow",
    "apiKey": "sk-5e8c1264f7b37b266ca43d42850434f7",
    "baseUrl": "https://apis.iflow.cn/v1/",
    "modelName": "qwen3-coder",
    "searchApiKey": "sk-5e8c1264f7b37b266ca43d42850434f7"
  }
  ~~~
